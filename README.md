# TaskMate
TaskMate adalah aplikasi untuk mengelola dan menjadwalkan berbagai pekerjaan rumah tangga secara harian.

Aplikasi dapat menyimpan data nama dan tugas yang akan dikerjakan, dihapus, dan tugas yang sudah diselesaikan. 
Masing-masing data tugas berisi:  1. Nama,
                                  2. Jenis tugas,
                                  3. Deskripsi tugas,
                                  4. Tingkat kesulitan (1 sampai 5),
                                  5. Estimasi waktu dalam menit, dan
                                  6. Kategori ruangan (input berupa angka).


Tugas Besar Matkul Algoritma dan Pemrograman 2 Telkom University Surabaya.
